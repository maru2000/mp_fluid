
# Multiphase Flow Datasets (FEM)
This repository contains MOOSE codes that generate datasets for multiphase flow simulations (Navier-Stokes)

Fork "mp_fluid" to create a new MOOSE-based application.

For more information see: [https://mooseframework.inl.gov/getting_started/new_users.html#create-an-app](https://mooseframework.inl.gov/getting_started/new_users.html#create-an-app)
